# hashD0G
This is a simple cryptographic tool that encrypts data using random parameters. It is based on a Cryptographically Secure Pseudo-Random Number Generator (CSPRNG).

### 🚀 Introduction
This crypto tool leverages fully random and secure parameters to encrypt and decrypt text. The core idea is to use randomness as the foundation of security.

### 📦 Installation
Follow the steps below to install and run the project:

1. Install the prerequisites using pip:
```python
pip install -r requirements.txt
```

2. Then run the main Python file:
```python
python hashdag.py
```
**Note: Make sure hashdag.py is located in the project directory.**
